# 🌍 Global Student Mobility Analysis

## 📌 Overview
This project analyzes international student mobility trends from 2019 to 2024 using a synthetic dataset. It focuses on understanding global distribution, COVID-19 impact, recovery patterns, and key factors influencing student destination choices.

---

## 🎯 Objectives
- Analyze global student distribution across countries and regions  
- Measure COVID-19 impact and post-pandemic recovery  
- Identify high-growth and high-volume destinations  
- Understand the role of academic fields and visa policies  

---

## 🛠 Tools & Technologies
- **Python (Pandas, NumPy)** – Data generation and preprocessing  
- **Power BI** – Data modeling, visualization, and dashboarding  

---

## 🧪 Data Generation
Since real-world data was limited, a synthetic dataset was created with realistic patterns:
- Pre-COVID baseline (2019)  
- Sharp decline in 2020 (pandemic impact)  
- Gradual recovery (2021–2024)  
- Country, field, and visa-based variations  
- Log-normal distribution for realistic student counts  

---

## 🧹 Data Cleaning
Performed in Jupyter Notebook:
- Handled missing values and inconsistencies  
- Standardized text fields (country, region, bloc)  
- Fixed data types and removed anomalies  
- Ensured logical consistency across years  

---

## 🧩 Data Modeling (Power BI)
- Created a **Fact Table** (student data)  
- Built **Dimension Tables** for:
  - Countries  
  - Fields of Study  
- Established **one-to-many relationships**  
- Enabled efficient filtering and accurate DAX calculations  

---

## 📊 Dashboard Highlights

### 🔹 Page 1: Global Overview
- Filled map of student distribution  
- Year-wise trend analysis  
- COVID impact and recovery KPIs  

### 🔹 Page 2: Competitiveness & Growth
- Recovery vs Pre-COVID comparison  
- Growth and market positioning (scatter plot)  
- Academic hub analysis (field → country)  
- Key drivers of growth  

---

## 📈 Key Insights
- Student mobility is concentrated in a few dominant countries  
- COVID-19 caused a sharp global decline, followed by uneven recovery  
- Recovery speed varies by region and policy environment  
- Certain countries act as specialized academic hubs  
- Visa flexibility and opportunities influence growth  

---

## 🚀 Conclusion
The project demonstrates how data analytics can uncover patterns in global education trends. It highlights the importance of policy, specialization, and resilience in shaping international student mobility.
